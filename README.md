# 🛠 Installation

Clone repository

```bash
git clone https://github.com/yourusername/angular-2048.git
```

Go to project

```bash
cd angular-2048
```

Install packages

```bash
npm install
```

Run application

```bash
ng serve
```

Open

```
http://localhost:4200
```

---

# 📦 Build

```bash
ng build
```

Production build

```bash
ng build --configuration production
```
